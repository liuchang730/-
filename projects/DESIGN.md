# DESIGN.md

## 气质与意象
如同微信跳一跳原版的简约手办感——一个安静的微缩世界，角色是小棋子，平台是散落的彩色小积木。背景随分数推进，从校门到各校园地标，仿佛在校园中打卡巡游。

## 配色方案
- 背景图：18张校园场景图，分数阈值触发叠化切换（1.2秒cross-fade）
- 平台：马卡龙色系（柔粉 #F8B4B4、薄荷绿 #A8D8B9、蜜桃橙 #F2C18D、薰衣草 #B8A9C9、天空蓝 #78C1C8、奶黄 #F7DC6F）
- 角色：骆驼图片(camel.png，透明背景3D卡通骆驼)，保留蓄力压缩/跳跃拉伸动画，未加载时回退为深炭色棋子
- 蓄力条：绿 → 黄 → 红渐变
- 智能体对话框：白色底(92%透明度)+淡灰边框，圆角药丸形，小三角指向右侧头像，深色文字 #333，多行自动换行（最大宽度220px），粗体15px
- 继续挑战按钮：暖橙色 #FF7832，圆角药丸形
- 通关文字：金色 #FFD700

## 字体排版
- 分数：无衬线粗体 36px
- 最高分：小字 14px，低透明度
- 通关标题：金色粗体 32px
- 智能体解说：粗体 15px，PingFang SC / Microsoft YaHei，深色文字 #333
- 继续挑战按钮：粗体 18px

## 页面结构
- 全屏 Canvas，无滚动条
- 顶部居中：当前分数 + 最高分
- 底部居中：蓄力进度条（仅蓄力时显示）
- 右下角：智能体头像(70px圆形+白色描边，3种情绪切换：常态/安慰/开心) + 左侧对话框气泡
- 2.5D 等轴测视角，平台沿左上/右上交替排列

## 交互与状态
- 长按蓄力：角色压缩，进度条填充
- 松开跳跃：抛物线弧线，角色拉伸
- 落地加分：+1 浮动文字（白色）
- 中心加分：等差数列加分（2,4,6...），金色浮动文字
- 掉落：角色下坠 → "继续挑战"按钮（保留分数继续）
- 200分通关：金色 "恭喜打卡校园成功！" + 最终得分
- 智能体解说触发（5种场景统一由「校园向导·奶酪」智能体输出）：game_start(首次加载)/perfect_land(精准落点)/high_score(高分50/100/150/200+连胜3+)/game_over(掉落)/landmark(地标切换)，流式输出，普通落地不触发
- 智能体头像3种情绪联动：常态(agent-avatar.png) / 安慰(agent-comfort.png，掉落时) / 开心(agent-happy.png，精准落点/高分时)，3秒后自动恢复常态

## 设计禁忌
- 不添加粒子特效、光效、阴影特效
- 不使用科技感渐变（蓝紫渐变等）
- 不添加多余装饰元素
- 不使用深色/暗黑风格
